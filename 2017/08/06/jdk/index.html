jdk 安装

1.安装条件

安装系统: centos7
安装版本: jdk-8u144-linux-x64.tar.gz

2.安装步骤:

1.卸载openjdk

卸载centos7自带的openjdk
–查看jdk版本: java -version
–查看有哪些jdk: rpm -qa|grep java
–卸载jdk: rpm -e –nodeps java-1.8.0-openjdk-1.8.0.102-4.b14.el7.x86_64(rpm -qa 查询出来的jdk版本)
rpm -e –nodeps java-1.8.0-openjdk-headless-1.8.0.102-4.b14.el7.x86_64(rpm -qa 查询出来的jdk版本带headless)

2.安装jdk
  1. 下载 jdk-8u144-linux-x64.tar.gz 并上传到 /opt 目录下 并解压 tar zxvf jdk-8u144-linux-x64.tar.gz
  2. 创建软连接: ln -s /opt/jdk1.8.0_144 /opt/jdk

  3. 配置环境变量

  4. source /etc/profile ← 使配置生效

3.校验jdk是否安装成功
